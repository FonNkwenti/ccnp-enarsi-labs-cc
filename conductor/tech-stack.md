# Technology Stack - CCNP ENCOR Lab Series

## Simulation & Virtualization
- **GNS3:** Primary network simulation platform.
- **Cisco IOS Images:** Optimized for Apple Silicon (M1/M2/M3) using `c3725` and `c7200` platforms.

## Automation & Testing
- **Python:** Used for lab automation and initial configuration deployment.
- **Pytest:** Framework for verifying lab configuration states and connectivity.

## Configuration & Data
- **YAML:** Used for defining lab baselines, device roles, and addressing schemes (e.g., `baseline.yaml`).
- **Shell Scripts:** Utility scripts for environment setup and log management.

## Documentation & Design
- **Markdown:** Standard format for all lab workbooks, instructions, and solutions.
- **Draw.io:** Primary tool for creating standardized topology diagrams (`.drawio`).
